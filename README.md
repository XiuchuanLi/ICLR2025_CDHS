This repository is the official implementation of [Recovery of Causal Graph Involving Latent Variables via Homologous Surrogates].

## Usage

Install kerpy

```(bash)
cd kerpy
python setup.py develop
cd ..
```
Run algorithm

```(bash)
cd algorithm
python main.py
```

## Requirement

python 3.8.19

numpy 1.24.3

scipy 1.10.1

pandas 2.0.3

matplotlib 3.7.2
